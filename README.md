# ThunderTaste

A recipe discovery website built as a Per Scholas capstone project. Users can browse recipes, search by ingredient or name, and view full recipe details.

## Features
- Recipe search and browse
- Recipe detail pages
- User login and account management
- Clean Bootstrap-based responsive UI

## Tech Stack
- **Frontend**: HTML, CSS, Bootstrap
- **Backend**: Java, Spring Framework, Thymeleaf
- **IDE**: IntelliJ / Eclipse

## Pages
- `index.html` — Home / recipe feed
- `recipe-search.html` — Search page
- `recipe-view.html` — Recipe detail
- `login.html` / `signup.html` — Auth pages
- `my-account.html` / `my-recipes.html` — User dashboard
